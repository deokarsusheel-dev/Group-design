## Project Info

**Project Name:** Group Design  
**Description:**  
This is a desktop-style UI application for a Bridge Design Module.  
The application allows users to input project details, validates inputs, auto-adjusts dependent fields, and displays reference values such as wind speed, seismic zone, and temperature based on selected locations.

This project is developed as part of a screening task for UI development.

---

## Features

- Tab-based layout with:
  - Basic Inputs tab (implemented)
  - Additional Inputs tab (placeholder)
- Project Location selection using predefined cities
- Automatic display of:
  - Wind Speed  
  - Seismic Zone and Factor  
  - Maximum and Minimum Temperature
- Input validation for:
  - Span  
  - Carriageway Width  
  - Skew Angle
- Modify Additional Geometry popup with:
  - Auto-updating dependent fields
  - Error messages for invalid values
- Bridge cross-section reference image display
